## 컴포넌트
- `index.js` : entry point. 최종 컴포넌트를 DOM에 render
- `App.js` : 모든 컴포넌트들의 root 컴포넌트

### Props
부모 컴포넌트에서 자식 컴포넌트로 내려주는 데이터
```jsx
function App() {
    return (
        <div>
            <MyComponent value={'test'}/>
        </div>
    )
}

function MyComponent(props) {
    return <div>{props.value}</div>
}
```

- 구조분해할당 구문 활용
- 특정 Props에 기본 값을 줄 수 있음.
- Props는 읽기 전용
